# procrastivity

A [Claude Code](https://docs.claude.com/en/docs/claude-code) plugin marketplace.

## Installation

Add this marketplace to Claude Code, then install any plugin from the list below.

From within Claude Code:

```
/plugin marketplace add procrastivity/claude-plugins
```

Or from the command line:

```
claude plugin marketplace add procrastivity/claude-plugins
```

To pull in newly added or updated plugins later, refresh the marketplace.

From within Claude Code:

```
/plugin marketplace update procrastivity
```

Or from the command line:

```
claude plugin marketplace update procrastivity
```

## Contributing

Plugin entries are defined in [`.claude-plugin/marketplace.json`](.claude-plugin/marketplace.json). Each plugin's
name, description, and version are sourced from its own `.claude-plugin/plugin.json`; this
`README.md` and the descriptions in `.claude-plugin/marketplace.json` are generated by
[`contrib/readme`](contrib/readme). After adding or changing a plugin entry, run:

```
contrib/readme
```
