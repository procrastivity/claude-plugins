# Contributing

Thanks for your interest in contributing to the **procrastivity** marketplace!

## How it works

Plugin entries are defined in [`.claude-plugin/marketplace.json`](../.claude-plugin/marketplace.json). Each plugin's
name, description, and version are sourced from its own `.claude-plugin/plugin.json`; the top-level
[`README.md`](../README.md) and the descriptions in `.claude-plugin/marketplace.json` are generated by
[`contrib/readme`](readme).

## Adding or changing a plugin

After adding or changing a plugin entry, regenerate the marketplace metadata:

```
contrib/readme
```

> [!IMPORTANT]
> Don't edit the generated sections of `README.md` or the plugin descriptions in
> `.claude-plugin/marketplace.json` by hand — run `contrib/readme` instead, since those
> sections are overwritten on each run.
